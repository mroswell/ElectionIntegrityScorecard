A note on methodology This is a framework, not an authoritative ranking. Scores populated here are illustrative placeholders drawing on commonly-cited sources (NCSL, Verified Voting, Brennan Center, state statutes). Before publication, every cell should be verified against primary sources and dated. Each criterion is scored 0–3:
  • 0 = absent or actively harmful
  • 1 = weak
  • 2 = adequate
  • 3 = strong protections
